# How to build your resume again

Becuase I know you forgot how to do this already

### Install
1. Ubuntu - `apt-get update; apt-get install texlive texlive-base`
2. Arch   - `pacman -S texlive-most`

### Do the thing
1. All - `pdflatex $FILE.tex`
